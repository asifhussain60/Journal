This file is the mandatory consultation reference before any writing pass.
Read this BEFORE opening a chapter. Every unused item must be evaluated for fit.
High relevance only. Never force a placement to use up an item.

---

## HOW TO USE THIS FILE

On every REFINE or CONTINUE pass:
1. Open this file first.
2. Find the target chapter's section below.
3. Evaluate every listed item against the passage being written.
4. Place only items that are genuinely strong — do not place just to reduce the unused count.
5. After placing, update quotes-library.txt "Used in" field AND mark the item in this index as PLACED.
6. Run auto_delta.py --save at session end.
